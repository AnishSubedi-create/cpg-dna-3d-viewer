CpG-A (ODN2216) · 20-mer
5'-ggGGGACGATCGTCgggggg-3' · Mixed PS/PO backbone
Four strands tetramerize via poly-G G-quadruplex caps (parallel topology, K⁺-stabilized at ~2.8 Å from O6). Palindromic PO center forms duplex stems. Forms ~100–200 nm globular aggregates. Binds HMGB1 → LLPS condensates → RAGE-SLP76 → inflammasome + IFN-α/β (pDC).

CpG-A · ODN2216

Structure: G-quadruplex tetramer

Backbone: Mixed PS/PO

Size: 20-mer · ~100–200 nm

Sensor: TLR9 (pDC endosome)

HMGB1: Direct binding → LLPS → RAGE → SLP76 → inflammasome

LLPS type: HMGB1·CpG-A condensates

Key output: IFN-α/β, IL-1β

CpG-B · ODN2006

Structure: Linear ssDNA

Backbone: Full phosphorothioate

Size: 24-mer · ~7–8 nm

Sensor: TLR9 (B cells, monocytes)

HMGB1: Weak — no LLPS; direct TLR9 dominates

LLPS type: None

Key output: IL-6, TNF-α, B-cell activation

Adenoviral dsDNA

Structure: B-form dsDNA, linear

Backbone: Phosphodiester (natural)

Size: ~36 kbp genome

Sensor: cGAS (cytosol); TLR9 weak

HMGB1: Protein VII sequesters HMGB1 (A-box) → blocks IFN-β

LLPS type: cGAS·dsDNA condensates → STING

Key output: IFN-β, ISG15, MX2

LLPS mechanistic comparison: CpG-A drives HMGB1 phase separation via the RAGE/SLP76 axis (PNAS 2025) — the G-quadruplex topology provides multivalent scaffolding unavailable to linear ODNs. Adenoviral dsDNA triggers a distinct cGAS·DNA LLPS → cGAMP → STING condensate pathway; adenovirus counteracts this using protein VII to physically sequester HMGB1 in chromatin via A-box interaction. CpG-B engages neither condensate pathway — it activates TLR9 through direct 1:1 endosomal engagement, driving NF-κB rather than IRF7/IRF3.
