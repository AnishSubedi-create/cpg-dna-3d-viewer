# cpg-dna-3d-viewer

## Description

This is a 3D viewer for cpg dna visualization.

## Live Demo

[▶ Live Demo](https://AnishSubedi-create.github.io/cpg-dna-3d-viewer/)

## Installation

Clone this repository and install dependencies.

## Usage

To run the viewer locally, execute the following command:

```bash
npm start
```

Visit the localhost link opened in your browser.
